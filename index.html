Calcula F y α para que el disco esté en equilibrio

Autor: EAG1964 | 2026 | Panel Principal | Licencia CC BY-NC-SA 4.0

Solución Paso a Paso

1. Equilibrio de Traslación

Σ →F = 0

Descomponiendo en el eje horizontal (X):

Σ →Fx = 50 N - 30 N - F = 0 → F = 50 - 30

F = 20 N (Hacia la izquierda)

2. Equilibrio Rotacional

Σ →M = 0

M1 + M2 + MF = 0

Observación de los ángulos: Toca cada fuerza para aislar su ángulo θ y su momento independiente:

r1·F1·sen(θ1) + r2·F2·sen(θ2) - rF·F·sen(θF) = 0

(0.10)(30)(1) + (0.20)(50)(0.5) - (0.75)(20)·sen(θF) = 0

3.0 + 5.0 - 15·sen(θF) = 0

15·sen(θF) = 8.0 → sen(θF) = 8/15

θF = arcsen(8/15) ≈ 32.2° (sentido horario)

Cálculo de α:

Como la fuerza F es horizontal, el radio forma con la horizontal izquierda un ángulo semejante a θF (32.2°).

Sumando los 90° del cuadrante superior izquierdo obtenemos α:

α = 90° + 32.2° = 122.2°
